Movie Quote API Documentation

Introduction

Welcome to the Movie Quote API documentation. This API provides access to the latest movie quotes, along with relevant information such as rating, author, author bio, and movie name. The API is free for the first month, and we're excited to have you on board.

Base URL

https://api.example.com/movie-quotes

Authentication

To access the Movie Quote API, you need to include your API key in the request headers. You can obtain your API key by registering on our developer portal.

Authorization: Bearer YOUR_API_KEY

Endpoints

1. Get Latest Movie Quotes

Endpoint: /quotes/latest

Method: GET

Description: Get the latest movie quotes.

Request

GET /quotes/latest

Response

{
  "quotes": [
    {
      "quote": "Your quote here.",
      "rating": 4.8,
      "author": "John Doe",
      "author_bio": "Renowned screenwriter",
      "movie_name": "Example Movie"
    },
    // Additional quotes...
  ]
}

2. Search Movie Quotes

Endpoint: /quotes/search

Method: GET

Description: Search for movie quotes based on keywords.

Request

GET /quotes/search?q=keyword

Response

{
  "quotes": [
    {
      "quote": "Your quote here.",
      "rating": 4.5,
      "author": "Jane Smith",
      "author_bio": "Award-winning director",
      "movie_name": "Another Movie"
    },
    // Additional quotes...
  ]
}

Error Handling

In case of an error, the API will respond with an appropriate HTTP status code and a JSON object containing details about the error.

Example:

{
  "error": {
    "code": 404,
    "message": "Not Found",
    "details": "The requested resource was not found."
  }
}

Rate Limits

To ensure fair usage, the API has rate limits. Free users are limited to 1000 requests per day.

Conclusion

Thank you for using the Movie Quote API. If you have any questions or need further assistance, please contact our support team at support@example.com.
